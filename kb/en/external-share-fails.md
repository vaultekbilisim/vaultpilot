# External share package fails to open

Use this when the offline share decrypter rejects a package.

## Checks

1. Confirm the package JSON was not edited.
2. Confirm the passphrase was copied from the one-time display.
3. Check the expiry time.
4. Check the maximum open count.
5. Confirm the browser supports Web Crypto.

## Error types

- Wrong passphrase.
- Tampered metadata.
- Expired package.
- Exhausted usage count.
- Unsupported package version.
