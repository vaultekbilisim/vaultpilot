# License read-only state

Use this when PassMan allows login and reading but blocks writes, user changes, sharing, pairing or update install actions.

## Checks

1. Open License.
2. Confirm trial or license expiry state.
3. Confirm active user count versus license capacity.
4. Paste a valid offline license code if provided by support.
5. Check that the license signature verifies successfully.

## Expected behavior

Read-only mode can keep login, secret viewing, audit review and backup export available while blocking write operations.
