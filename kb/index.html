Fast operator answers, without leaking secrets.

Use these public-safe entries to diagnose common installation, update, AD sync, extension, certificate, audit and license states.

MSI

Installation fails

Check admin context, port ownership, service state and installer logs.

 Update

Update stays around 76%

Understand service restart reconciliation and verbose MSI log review.

 AD Agent

Service cannot connect

Verify URL reachability, bind username format, redacted logs and repair mode.

 Extension

Pairing remains pending

Review pair code lifetime, username, unlocked vault state and device approval.

 HTTPS

Certificate warning

Check host/SAN match, PFX/P12 path, server ACL and firewall route.

 Audit

Audit chain is partial

Inspect related events with advanced filters and treat unexplained breaks as incidents.

 License

Read-only state

Review trial expiry, license signature, active user count and activation status.

 Share

External package fails

Separate wrong passphrase, expired package, exhausted usage and tampered metadata.

 Safety

What can I send to support?

Use redacted logs and metadata only; never send secrets, tokens, DB files or private keys.
