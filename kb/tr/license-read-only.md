# Lisans read-only durumu

PassMan login ve okuma izni veriyor ama yazma, kullanıcı değişikliği, paylaşım, pairing veya update install aksiyonlarını engelliyorsa bunu kullanın.

## Kontroller

1. Lisans ekranını açın.
2. Trial veya license expiry durumunu doğrulayın.
3. Aktif kullanıcı sayısını license capacity ile karşılaştırın.
4. Support tarafından verilen geçerli offline lisans kodunu yapıştırın.
5. Lisans imzasının doğrulandığını kontrol edin.

## Beklenen davranış

Read-only mode login, secret viewing, audit review ve backup export erişimini korurken yazma işlemlerini engelleyebilir.
