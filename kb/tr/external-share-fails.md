# Dış paylaşım paketi açılamıyor

Offline share decrypter paketi reddediyorsa bunu kullanın.

## Kontroller

1. Package JSON dosyasının değiştirilmediğini doğrulayın.
2. Passphrase değerinin one-time display ekranından doğru kopyalandığını kontrol edin.
3. Expiry zamanını kontrol edin.
4. Maksimum açma sayısını kontrol edin.
5. Tarayıcının Web Crypto desteklediğini doğrulayın.

## Hata tipleri

- Yanlış passphrase.
- Değiştirilmiş metadata.
- Süresi dolmuş paket.
- Kullanım hakkı bitmiş paket.
- Desteklenmeyen paket versiyonu.
