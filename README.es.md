## L&iacute;mite P&uacute;blico de Seguridad

No agregue secretos reales, master passwords, contenido de vault, API keys, agent tokens, material privado de licencia, private keys, PFX/P12, bases de datos, backups, raw logs ni screenshots con datos reales de clientes.

Use placeholders como &lt;VAULTPILOT_URL&gt;, &lt;SERVER_HOST&gt;, &lt;AGENT_ID&gt;, &lt;AGENT_TOKEN&gt;, &lt;LICENSE_CODE&gt; y &lt;REDACTED&gt;. Antes de abrir un issue p&uacute;blico, revise [public issue redaction](kb/en/public-issue-redaction.md).
