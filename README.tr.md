# VaultPilot Enterprise Vault Console - Türkçe Giriş

Diller: [English](README.md) | [Türkçe](README.tr.md) | [Deutsch](README.de.md) | [Español](README.es.md) | [Français](README.fr.md) | [Português (Brasil)](README.pt-BR.md)

Bu sayfa Türkçe hızlı giriş noktasıdır. Tam Türkçe operatör akışı için [docs/tr/README.md](docs/tr/README.md) sayfasını, ana herkese açık repo haritası için [README.md](README.md) dosyasını kullanın.

VaultPilot, Windows Server üzerine MSI ile kurulan ve kullanıcı tarayıcılarından erişilen, kendi sunucunuzda çalışan sıfır bilgi temelli parola ve sır yöneticisidir. Bu herkese açık dokümantasyon reposu kaynak kod deposu değildir; yayın yönlendirmesi, operatör dokümanı, bilgi bankası, güvenlik politikası, gizlilik metni ve destek sınırı için kullanılır.

## Herkese Açık Güvenlik Sınırı

Bu repoya gerçek secret, ana parola, vault içeriği, API anahtarı, agent token, lisans private materyali, private key, PFX/P12 dosyası, veritabanı, yedek, ham log veya gerçek müşteri verisi içeren ekran görüntüsü eklemeyin.

Herkese açık örneklerde `<VAULTPILOT_URL>`, `<SERVER_HOST>`, `<AGENT_ID>`, `<AGENT_TOKEN>`, `<LICENSE_CODE>` ve `<REDACTED>` gibi yer tutucular kullanın. Herkese açık issue açmadan önce [public issue redaction KB](kb/tr/public-issue-redaction.md) sayfasını kontrol edin.

## Katkı

Doküman, KB, link, çeviri veya herkese açık paylaşım için güvenli ekran görüntüsü düzeltmesi açmadan önce [CONTRIBUTING.md](CONTRIBUTING.md) dosyasını okuyun. Release binary'leri, private kaynak kod, imzalama materyali veya müşteri kanıt paketleri bu repoya ait değildir.
